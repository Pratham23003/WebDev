# GitCard-JS

A lightweight, customizable JavaScript widget for displaying GitHub profile cards on any website.

---

##  Demo
See it live: [pratham23003.github.io/GitCard-JS/](https://pratham23003.github.io/GitCard-JS/)

---

##  Features

- Fetches **real-time GitHub profile data** using vanilla JavaScript and `XMLHttpRequest` (no dependencies).  
- Displays user’s **name, avatar, bio, repositories**, and more.  
- **Fully responsive and customizable** styles (CSS & JS).  
- No frameworks required — works with just **JavaScript, CSS, and HTML**.

---

##  How It Works

1. Utilizes the **XMLHttpRequest API** to request user data directly from the **GitHub public API**.  
2. Processes and displays the information **dynamically** on the web page.
